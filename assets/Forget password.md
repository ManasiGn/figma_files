# Forget Password Prototype

## Overview
This project showcases a **Forget Password user flow** designed and prototyped using Figma. The prototype focuses on creating a smooth and intuitive password recovery experience for users.

## About the Project
The design demonstrates how users can securely reset their password through a guided process that includes verification steps, clear navigation, and interactive feedback. The prototype emphasizes usability, simplicity, and user-centered design to ensure a seamless account recovery experience.

## Features
- Password recovery user flow
- Clear step-by-step interaction design
- Interactive screen transitions
- User-friendly navigation
- High-fidelity Figma prototype
