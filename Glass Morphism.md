# Glass Morphism Prototype

## Overview
This project showcases a **Glass Morphism UI design prototype** created using Figma. The design demonstrates a modern visual style that uses transparency, blur, and layered elements to create a frosted glass effect and a sense of depth.

## About the Project
Glass morphism is a popular UI design trend characterized by **semi-transparent surfaces, background blur, and soft shadows**, giving elements a glass-like appearance. :contentReference[oaicite:0]{index=0}  

This prototype explores how glass morphism can enhance visual hierarchy, improve focus on key elements, and create a clean, premium user interface. The design uses layered components and subtle motion to simulate depth and separation between UI elements.

## Features
- Glass morphism (frosted glass) effect
- Transparent and blurred UI elements
- Layered design for depth and hierarchy
- Modern and minimal interface
- Interactive Figma prototype
