# Movie Booking App – Figma Prototype

## Overview
This project showcases a **Movie Booking App UI/UX prototype** designed using Figma. The prototype demonstrates the user journey of browsing a movie, selecting a show date and time, and choosing seats for booking.

The goal of this project is to practice **user flow design, visual hierarchy, and interactive prototyping** while creating a smooth and intuitive movie ticket booking experience. Modern movie booking interfaces typically guide users through steps such as selecting a movie, choosing a showtime, selecting seats, and confirming the booking. :contentReference[oaicite:0]{index=0}

## Features
- Movie browsing interface
- Date and showtime selection
- Interactive seat selection layout
- Step-by-step booking flow
- High-fidelity UI design
- Interactive Figma prototype

## How to Use the Prototype
Follow these steps to experience the main booking flow in the prototype:

1. Click on **Doctor Strange** movie.
2. Select the date **Sat 23**.
3. Choose the showtime **18:00**.
4. On the seat selection screen, choose **Seat 9 and Seat 10** from the **second row from the front**.

This sequence demonstrates the core interaction flow of the movie booking process.
