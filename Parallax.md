# Parallax Effect Prototype

## Overview
This project showcases a **parallax scrolling effect** designed and prototyped using Figma. The interaction creates an illusion of depth by moving background and foreground elements at different speeds during scrolling, resulting in a dynamic and immersive visual experience.

## About the Project
The prototype demonstrates how motion design principles can enhance user engagement and visual storytelling within a user interface. By using layered components, scrolling interactions, and Figma’s prototyping features, the design simulates real-world motion in a 2D digital environment.

## Features
- Parallax scrolling interaction
- Layered UI elements for depth simulation
- Smooth scrolling animation
- Interactive Figma prototype
